# Furuta (Rotary) Inverted Pendulum  
**Modeling, Control, and Hardware-in-the-Loop Implementation**

## Overview
This project presents the **control, simulation, and experimental validation** of a **Furuta (rotary) inverted pendulum**, a classic nonlinear and inherently unstable control system.

The system dynamics are taken directly from a well-established reference in the literature and linearized around the upright equilibrium. Control is achieved using **modern control techniques**, primarily the **Linear Quadratic Regulator (LQR)**. Validation is performed through **MATLAB/Simulink simulations**, **Simscape Multibody modeling**, and **Hardware-in-the-Loop (HIL)** experiments using an Arduino-based setup.

---

## Key Features
- **System dynamics adopted from literature** for accurate and validated modeling  
- **Linearization** around the unstable upright equilibrium  
- **LQR controller design** with performance-oriented tuning  
- **Parameter estimation** of the DC motor using experimental data and optimization methods  
- **Swing-up control** based on the total energy approach  
- **Simscape Multibody model** generated by exporting CAD geometry (XML) and using the SimMechanics interface  
- **Hardware-in-the-Loop (HIL)** implementation in Simulink  
- **Real-time Simulink dashboard** for monitoring system states and control signals  

---

## System Components
- Geared DC motor with encoder  
- Rotary arm and pendulum rod  
- Arduino Uno (low-level control and data acquisition)  
- Motor driver and power electronics  
- MATLAB/Simulink and Simscape Multibody
